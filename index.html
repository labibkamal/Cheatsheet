The Position Property

The position property indicates how an element is positioned in relation to all other elements of a webpage. With it the user can control how and where an element will be displayed on a webpage in relation to other elements.

There are five different values that can be assigned to the position property.

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky
Name Description Top, Right, Bottom, Left, z-index compatible
Static Element positioned according to normal flow of page. Default value for elements. No
Relative Element positioned in relation to its original position on the page. Yes
Fixed Element is removed from normal flow and positioned relative to the viewport, appearing in the same place when scrolling. Yes. Position is controlled by these.
Absolute Element is positioned relative to it's nearest ancestor, or it's containing block Yes
Sticky Element is positioned based on user's scroll position. Toggles between relative to fixed when an offset position is met in viewport. Yes. Used to positoin where element will be positioned after scrolling.

Important Notes

Viewport: The visible area of a window screen that is currently being displayed. While the entire page may be rendered, the user sees the content in the viewport.
