The Position Property

The position property indicates how an element is positioned in relation to all other elements of a webpage. With it the user can control how and where an element will be displayed on a webpage in relation to other elements.

There are five different values that can be assigned to te position property.

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky
Name Description Top, Right, Bottom, Left, z-index compatible
Static
Relative
Fixed

Static

HTML elements with position static is positioned according to the normal flow of the page. This is the default value for positioning for all elements. Elements with the position set to static are not affected by top, right, bottom, left, or z-index properties.

Relative

HTML elements with position relative is positioned, using the top, right, bottom, and left properties, in relation to its original position in accordance to the flow of the page. IMPORTANT: Other elements will bot be able to fit into the gap left by the element.
